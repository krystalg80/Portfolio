## About

This portfolio showcases my work as a Full Stack Software Engineer specializing in React, Node.js, TypeScript, and AI-powered applications. It includes my projects, work experience, education, skills, and contact information.

---

## Features

- **Terminal boot screen** — Typing animation on load before transitioning into the portfolio
- **Dark hero section** — Deep forest green hero with headshot, social links, and CTA
- **Floating leaf animation** — Subtle ambient background effect
- **Project cards** — Flip-on-hover cards with tech stack tags, descriptions, and links
- **Work experience carousel** — Timeline-style experience section
- **Education section** — Two-column card layout
- **Skills section** — Categorized front-end, back-end, and tools
- **Contact section** — Card grid with email, LinkedIn, GitHub, and location links
- **Fully responsive** — Mobile, tablet, and desktop layouts
